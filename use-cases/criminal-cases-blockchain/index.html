USE CASE

Streamlining small criminal cases

Blockchain has opened up countless opportunities for organizations to collaborate smarter and transparent. When novel technologies are combined, actual efficiency surfaces.

Involved parties

Police and District Attorneys.

Business challenges

Many stakeholders

At the moment, multiple different government bodies take part in this repetitive process, working with unstructured data.

Lots of paperwork

With many stakeholders that produce unstructured data, much paperwork is created on these repetitive processes using faxes, email, and phone calls.

Existing systems

We use tailored IT-systems on these processes in an organization, and replacing them would be a massive burden with associated costs.

Solution overview

The Process

These processes are incredibly cumbersome and provided a lot of unstructured data, even through using ancient fax machines. LTO Networks' distributed business logic allows for all government bodies to trust the validity of the data.

Data sharing

with LTO Network's distributed data processing, data is exchanged over linked (legacy) systems that run at the stakeholders' offices. The implementation of blockchain shouldn't require the replacement of otherwise-working systems for the sake of it.

Citizen engagement

Based on the state of the workflow, citizens (victims, perpetrators), can receive filtered information about the process through their (mobile) DigiD app, eliminating the need for status update calls with the Districts Attorney's office.

Book a demo

Advantages using LTO Network

Decentralized workflows

Using LTO Network's technology, we run small criminal cases' paperwork on the blockchain, based on the predefined logic at the core of each process. All parties involved can participate in the process using their systems, as process data is automatically exchanged between these systems.

Adding AI

IBM Watson will further reduce the manual labor required in a ZSM-process, using a cognitive computing system to instantly understand questions and respond with the correct answers and verdicts, based on the input of stakeholders.

Meaningful tasks

Since LTO Network takes away a significant part of the administrative burdens, this way of working frees up knowledge and expertise for other essential functions that cannot be performed without human assistance (yet).
